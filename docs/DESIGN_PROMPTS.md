# TrailKit Design Prompts

Use these prompts in ChatGPT Images or design tools to generate polished assets. Keep the output original and do not reference existing company logos.

## Logo Mark

Create an original logo mark for a startup called TrailKit. The mark is a simple trail/path line made of three connected nodes representing click, install, and revenue. Style: modern mobile infrastructure, precise, lightweight, trustworthy, dark ink background, electric cyan first node, mint second node, warm amber third node, clean vector geometry, no text, transparent background option, not a copy of any existing logo.

## Transparent-Background Logo

Create a transparent-background horizontal logo for TrailKit with a three-node attribution path mark and a clean modern wordmark. The wordmark should feel technical but founder-friendly. Use deep ink text, electric cyan accent on "Kit", mint and amber accents in the mark. No copyrighted templates, no stock symbols, no gradients that overpower readability.

## Wordmark

Design a custom-feeling TrailKit wordmark using a clean geometric sans style. Make it precise, modern, and readable at small sizes. Use subtle weight contrast between "Trail" and "Kit" without using a paid font. Avoid extreme letter spacing. Provide dark and light background versions.

## App Dashboard Hero Visual

Create a polished product hero visual for TrailKit, a mobile attribution dashboard. Show cards for total clicks, installs, subscriptions, revenue, estimated payouts, a source-level table, and a small attribution flow. Dark ink UI, electric cyan, mint, and amber accents, thin grid lines, founder-friendly SaaS analytics, realistic but not cluttered.

## Smart Link Flow Diagram

Create a clean diagram showing the TrailKit smart link flow: creator/campaign/email/QR link -> click -> app store routing -> first open -> revenue event -> payout estimate. Use node-link paths, dark background, cyan/mint/amber signal colors, concise labels, technical infrastructure feel.

## Attribution Path Visual

Create an abstract attribution path visual for a mobile app growth tool. Show many small channel nodes converging into install, subscription, revenue, and payout nodes. The image should feel precise, privacy-conscious, and lightweight. Avoid depicting surveillance, fingerprinting, or people profiles.

## Abstract Landing Page Background

Create a dark, subtle infrastructure background for TrailKit. Use thin grid lines, faint path curves, small glowing nodes, and restrained cyan, mint, and amber highlights. The background should support readable white text and not look like generic neon cyberpunk.

## Product Mockups

Create a set of clean product mockups for TrailKit: dashboard overview, smart links table, link detail with event timeline, partner payout summary, and settings integration page. Use dark UI, 8px cards, crisp typography, realistic metrics, and mobile-growth attribution language.
